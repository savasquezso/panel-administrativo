Fruver Dashboard
¡Bienvenido al proyecto Fruver Dashboard! Este es un sistema desarrollado para administrar la gestión de proveedores, ventas y otros elementos clave para la operación de una tienda de frutas y verduras. El dashboard proporciona una interfaz amigable para visualizar, analizar y manejar la información de manera eficiente.

Características
Gestión de Ventas: Permite registrar, modificar y eliminar ventas, con análisis detallado de ganancias diarias, semanales y mensuales.
Gestión de Proveedores: Administra información de proveedores, incluyendo creación, actualización y eliminación de registros.
Interfaz Intuitiva: Visualización de datos a través de gráficos y cuadros informativos para facilitar la toma de decisiones.
Filtros y Búsquedas: Opciones avanzadas para filtrar y buscar registros de ventas y proveedores.
Seguridad: Manejo de roles y autenticación de usuarios.
Tecnologías Utilizadas
Lenguaje: PHP
Base de Datos: MySQL (usando XAMPP)
Frontend: HTML, CSS y JavaScript
Frameworks/Librerías: Bootstrap para el diseño responsivo

Instalación
Clona el repositorio:

bash
Copiar código
git clone https://github.com/tu-usuario/dashboard_fruver.git
cd dashboard_fruver
Configura tu base de datos:

Crea una base de datos llamada fruver_db usando tu servidor MySQL.
Importa el archivo database.sql proporcionado (si aplica).
Configura el entorno:

Edita el archivo de configuración de la base de datos (config.php) con tus credenciales locales.
Inicia tu servidor local:

Utiliza XAMPP para arrancar Apache y MySQL.
Abre el proyecto en tu navegador:

arduino
Copiar código
http://localhost/dashboard_fruver
Uso
Proveedores: Gestiona tus proveedores desde la sección de administración de proveedores, donde puedes crear, editar o eliminar registros.
Ventas: Realiza operaciones sobre registros de ventas con la posibilidad de generar análisis y reportes.
Estadísticas: Visualiza gráficos y resúmenes de datos clave para tomar decisiones más informadas.
